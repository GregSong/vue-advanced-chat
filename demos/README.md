# 功能演示集合

`demos` 目录下按功能拆分了多个示例，便于快速了解组件的各项能力。

| 序号 | 功能             | 说明                 |
| ---- | ---------------- | -------------------- |
| 01   | 文本格式展示     | `FormatMessage` 解析各种文本标记 |
| 02   | 表情选择器       | 独立使用表情选择组件 |
| 03   | 附件消息         | 图片、语音等文件展示 |
| 04   | 编辑与回复       | 消息编辑和回复逻辑   |
| 05   | 用户标签         | 输入 @ 选择并展示标签 |
| 06   | 消息状态         | 已读、新消息及系统消息 |
| 07   | 主题切换         | 亮色与暗色主题演示   |

每个子目录都包含 `README.md` 和示例代码，可直接作为二次开发的参考。
